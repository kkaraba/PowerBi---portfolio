🍕 Pizza Sales BI Dashboard Power BI Report για Ανάλυση Πωλήσεων Πιτσαρίας

📌 Περιγραφή Ένα ολοκληρωμένο Power BI dashboard για την ανάλυση πωλήσεων μιας πιτσαρίας, βασισμένο σε δεδομένα από το Kaggle. Το report περιλαμβάνει:

Executive Dashboard: KPIs, μηνιαία τάση πωλήσεων, ανάλυση ανά προϊόν

Heatmap Analysis: Χάρτης θερμότητας με πωλήσεις ανά ώρα/ημέρα για βελτιστοποίηση προσωπικού

🔍 Key Insights Από την ανάλυση προκύπτουν ενδιαφέροντα ευρήματα:

Εύρημα Τιμή 🏆 Καλύτερος μήνας Ιούλιος (€72.557) 📉 Χειρότερος μήνας Οκτώβριος (€64.027) 🕐 Peak hours 12-1μμ & 5-6μμ 📅 Πιο πολυσύχναστη μέρα Παρασκευή (8.242 παραγγελίες) 🍕 Top πίτσα BBQ Chicken (€42.768) 📏 Δημοφιλέστερο μέγεθος Large (46% των εσόδων) 🛠 Τεχνολογίες Power BI Desktop DAX (Data Analysis Expressions) Power Query (καθαρισμός δεδομένων) Dataset: Pizza Sales Dataset από Kaggle
